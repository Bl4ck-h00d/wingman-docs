# wingman-docs

Wingman repository is not open source yet. This repository serves as a high level overview of Wingman Architecture.

## Tech Stack

### Client
* Typescript
* React.js
* Redux
* SASS

### Backend
* Typescript
* Node.js
* RabbitMQ
* PostgreSQL

### Deployment
* AWS EC2 & S3
* NGINX
* DOCKER
